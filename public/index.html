Timestamp Microservice

Pass a string as a parameter -- if the string contains either a Unix timestamp or a natural language date, the Timestamp Microservice returns both the Unix timestamp and the natural language form of that date; if not, the Timestamp Microservice returns null for both properties



Example:

http://zhinonihz-timestamp.herokuapp.com/March 20, 2016

http://zhinonihz-timestamp.herokuapp.com/1458446400

Example output:

{"unix":1458446400,"natural":"Sun, 20 Mar 2016 04:00:00 GMT"}
